# WEVE DESIGN 홈페이지

Next.js와 Sanity Studio로 만든 WEVE DESIGN 홈페이지입니다.

## 실행

```bash
npm run dev
```

홈페이지: http://localhost:3000  
관리 화면: http://localhost:3000/studio-weve-3891

## 포트폴리오 관리

관리 화면에서 로그인한 뒤 아래 순서로 운영하면 됩니다.

1. `공간 카테고리`에서 아파트, 주방, 욕실 같은 분류를 만듭니다.
2. `포트폴리오`에서 새 프로젝트를 추가합니다.
3. 제목, 카테고리, 지역, 면적, 설명, 대표 이미지를 입력합니다.
4. 메인에 먼저 보여주고 싶은 프로젝트는 `메인에 대표 사례로 노출`을 켭니다.
5. 임시 저장만 하고 홈페이지에 숨기려면 `홈페이지에 노출`을 끕니다.
6. `Publish`를 누르면 홈페이지에 반영됩니다.

## 관리자 주소와 보안

공개 홈페이지에는 관리 화면 링크가 없습니다. 기존 `/admin` 주소는 404로 닫았고, 현재 관리 화면은 `/studio-weve-3891`입니다.

주소를 숨기는 것은 보조 장치입니다. 실제 접근 제한은 Sanity 로그인과 프로젝트 권한으로 관리해야 합니다. Sanity 계정에는 운영자만 초대하고, 퇴사자나 외부 작업자는 즉시 권한을 제거하세요.

## 상담 메일 설정

상담 신청 메일 발송에는 아래 환경변수가 필요합니다.

```bash
RESEND_API_KEY=your_resend_api_key
CONSULTATION_TO_EMAIL=ehogh1@gmail.com
```

배포 환경에서도 같은 환경변수를 등록해야 상담 폼이 정상 동작합니다.
